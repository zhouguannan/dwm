# dwm

kevin's dwm is based on @theniceboy

dwm is an extremely fast, small, and dynamic window manager for X.

## requirements

In order to build dwm you need the Xlib header files.

## how to install dwm 



```bash
cd dwm/
sudo make clean install
```

## how to run dwm

Add the following line to your .xinitrc to start dwm using startx(after install startx):

```
exec dwm
```
